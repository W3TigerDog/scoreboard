# 🏀 Scoreboard App

A simple interactive basketball scoreboard built with HTML, CSS, and JavaScript.

## 🌐 Live Demo

👉 https://scoreboardbyt3tigerdog.netlify.app/

## 📌 Overview

This project demonstrates fundamental frontend development concepts including:

- DOM manipulation
- Event-driven programming
- State management
- UI updates in real time

It was built as part of my journey toward becoming a frontend and Web3 developer.

---

## 🚀 Features

- Add 1 / 2 / 3 points
- Prevent negative scores
- Separate home and guest tracking
- Instant UI updates
- Clean minimal layout

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Netlify (Deployment)

---

## 📚 Key Concepts Practiced

- JavaScript functions
- Working with `document.getElementById`
- Handling click events
- Updating DOM elements dynamically
- Basic application state logic
